ParaX Pro - Complete macOS Installer package

Included:
- ParaX Pro Mac Installer.command (interactive installer)
- Files/ParaX Pro.jsxbin
- Files/ParaX Pro Header Logo.png
- Files/PU_Settings_v11.xml
- mac-pkg/ (full .pkg builder with resources/scripts/distribution)

Interactive install (easy):
1) unzip this folder on macOS
2) open Terminal here
3) chmod +x "ParaX Pro Mac Installer.command"
4) ./"ParaX Pro Mac Installer.command"

Build native .pkg (advanced):
1) cd mac-pkg
2) chmod +x build-mac-pkg.sh scripts/postinstall
3) ./build-mac-pkg.sh
